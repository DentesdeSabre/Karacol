# Karacol & Linha — Next.js 14 App Router starter

Mobile-first site with App Router, TypeScript, Tailwind, Supabase storage/auth and shadcn/ui-ready components.

Quick start

1. Install dependencies:

```bash
npm install
```

2. Copy `.env.example` to `.env.local` and fill values.

3. Start dev server:

```bash
npm run dev
```

Supabase

- Run `sql/schema.sql` in Supabase SQL editor.
- Create storage bucket `karacol-assets` (private).
